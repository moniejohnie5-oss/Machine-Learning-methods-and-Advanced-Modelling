# Machine Learning Methods & Predictive Modeling

This repository contains independent computational scripts and projects focused on the design, optimization, and evaluation of machine learning algorithms. The focus here is on bridging statistical mathematical theory (such as gradient descent, linear algebra, and probability) with clean, scalable Python and R code.

---

## 🧠 Algorithmic & Mathematical Focus

Every project in this repository explores the underlying mathematical structures of modern data models:
* **Optimization:** Minimizing loss functions via Gradient Descent.
* **Probability & Statistics:** Maximum Likelihood Estimation (MLE) and statistical significance testing.
* **Linear Algebra:** Matrix operations, eigenvectors, and high-dimensional space transformations.

---

## 📂 Repository Structure & Projects

### 📂 /supervised-learning
* **Linear Regression from Scratch:** Implementation of multi-variable linear regression using Ordinary Least Squares (OLS) to solve the normal equation.
* **Logistic Regression Classifier:** Binary classification models predicting outcomes based on log-odds ratios, evaluated using ROC-AUC and Confusion Matrices.
* **Regularization Techniques:** Scripts applying Ridge (L2) and Lasso (L1) regression to control model overfitting and perform automated feature selection.

### 📂 /ensemble-methods
* **Random Forest Optimization:** Tree-based ensemble structures designed to evaluate feature importance and minimize prediction variance across highly non-linear datasets.
* **Gradient Boosting Implementations:** Utilizing iterative residual learning frameworks to maximize predictive accuracy on complex tabular data.

### 📂 /evaluation-metrics
* Comprehensive statistical frameworks to analyze model behavior, including Precision-Recall curves, F1-Score calibration, and cross-validation variance tracking.
